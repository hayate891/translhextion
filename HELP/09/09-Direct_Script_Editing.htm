Thingy IME - Direct Script Editing


To directly edit a the script within a file make sure you have a table file loaded and that read-only mode is disabled.  Place the cursor on the byte where you'd like to start editing, use the <INS> button to select the desired mode (either insert or overwrite), and press <ENTER>.  The Thingy IME window will appear.  Start typing.  Your keystrokes will be encoded using the loaded table as you type.  Every 100 bytes the IME window will update the file with your string and return to the IME window to allow you to keep typing.  You can stop editing by pressing <ENTER> again (this will update the file with whatever is in the edit-box and close the IME window).  If you accidentally open the IME window and don't want to edit the script just press <ENTER> when the edit box is empty and the IME will close.
