Optimize Script... (command Script/Optimize Script...)


This command parses a script file through a Thingy Table making the best possible use of DTE (Dual Tile Encoding), MTE (Multiple Tile Encoding) and multi-byte values for the words in the script. It outputs another script file in "Separated-byte Format" that, if DTE, MTE, or multi-byte values are available and applicable, should take up less bytes when reinserted into a file.

This can be used to convert a script that is not in Separated-byte Format to one that is.
