Japanese


This command enables or disables the Japanese font which displays Hiragana and Katakana characters instead of their ascii equivalents when "Thingy view" is enabled. If the correct encoding of the loaded table is not selected, the output will not be accurate.

Shift-JIS
This option specifies that the table will be interpreted using Shift-JIS format when displaying Japanese characters.

EUC-JIS
This option specifies that the table will be interpreted using EUC-JIS format when displaying Japanese characters.
