Paste (command Selection/Paste)


This either inserts or overwrites data from the clipboard starting at the current offset. Choose a radio button for the writing mode wished for. You can also choose how many times the current data is written.

If "Paste coded binary values as text" is unchecked, all data in the clipboard is considered to contain special byte-code syntax, so <ld:150> will not be written as the text "<ld:150>" but as a longword value of 150. Check the checkbox if you want to paste as text. The binary mode (Little or Big-endian) and character set (ANSI or OEM) is used as set in the options.
