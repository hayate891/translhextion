Insert decimal value... (command Insert/Insert decimal value...)


With this you can overwrite data with an integer value of either byte, word or longword size. The start offset is initialized with the current offset, in hexadecimal notation. Note that if you change the start offset, use hexadecimal notation and prefix with "x". You can also choose the number of times to overwrite or insert and the size of the value. The values are initialized with the current offset, the value of the current offset, byte size and 1 time.
