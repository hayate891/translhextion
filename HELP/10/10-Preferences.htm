Preferences... (command Options/Preferences...)


Here you can choose:
- To automatically adjust the number of bytes per line of hexdump or specify a value if Auto is disabled
- The number of digits for the offset length
- How to display the values at the caret position (signed or unsigned integer values)
- The character set used in displaying and writing characters (ANSI - Windows, or OEM - DOS)
- The font size in points
- The left enclosure & right enclosure (**) for Separated-byte Format as well as if the format should be used as the default for dump/replace operations

- If read-only mode should be automatically set after opening files
- To integrate/uninstall Translhextion into the Windows Explorer Context menu
- The name of the text editor that will be called by the Edit/Open in text editor command.
- The name of the Internet browser that will be called in the About dialog

(**) - These enclosure characters MUST be uniformly used during script operations. I.E. if you dump a script with one set of enclosure characters you must also replace that script using the same set!!!!
